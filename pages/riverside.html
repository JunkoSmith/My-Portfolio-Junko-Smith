    Designing and Developing a Responsive Website Project
    Designing and Developing the Riverside Studio Website Designing and Developing the Riverside Studio Website Designing and Developing the Riverside Studio Website Designing and Developing the Riverside Studio Website
    Overview of the Project

    This website was created to promote a local photography studio called Riverside Studio, which offers a diverse range of photographic services.

    Originally created using a UCD, UI, and SEO approach, the design aimed to increase user traffic and profit for the studio, as well as provide a valuable resource to potential users of their services.

    Following feedback that deeper user analysis and further SEO would make the website more valuable, I undertook a full redesign applying SEO and UX design principles.

    This project was created as part of a college assignment based on a fictional client brief, designed to simulate a real-world web design scenario.
